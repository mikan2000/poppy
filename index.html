HTML(Hyper Text Markup Language)

HTML(Hyper Text Markup Language)

简介

Hyper Text Markup Language是超文本标记语言

它是由很多标签组成 ，通常都是成对出现，如：

<!DOCTYPE html>
<html>
    <head>
        <title>Hello world!</title>
    </head>
    <body>
        <p>Hello world!</p>
    </body>
</html>
