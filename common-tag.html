HTML(Hyper Text Markup Language)

常用标签

block element 与 inline element

HTML中元素大多数都是块级元素（block element）或行内元素（inline element）。

默认情况下，block element前后增加断行，这些元素的默认宽度会填满父容器。而inline element不会造成断行。

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <style type="text/css">
        div, p, h5, span, a, button {
            background-color: #BDC0BA;
        }
    </style>
</head>
<body>
    <div>div</div>
    <p>p</p>
    <h5>h5</h5>
    <span>span</span>
    <a href="#">a</a>
    <button>button</button>
</body>
</html>

效果如图，其中div、p、h5是block element，span、a、button是inline element。

块级元素（block element）

  • div

    div是一个通用型的流内容容器，它语义上不代表任何内容，通常用div来布局。 <section>, <article>, <nav>, <header>, <footer>是语义化的div，当然也是block element。

     <div class="container"></div>
  • h1, h2, h3, h4, h5, h6

    页面上内容的标题，h1是一级标题，以此类推。h1在一个网页上只能出现一次。

     <h1>一级标题</h1>
 <h2>二级标题</h2>
  • p

    表示段落的标签。在调整段落之间间隙时要使用css margin属性，不要插入空段落或者<br>。

     <p>第一个段落</p>
 <p>第二个段落</p>
  • form

    用于创建html提交表单，可以包含<input>元素。其中action属性是表单所提交到的url地址。method是提交方式，它可以是post或get等。

     <form action="#" method="get">
     <input type="text" name="name" />
     <input type="submit" value="save" />
 </form>
  • ul, li, ol, dl, dt, dd

    ul, ol, li表示一组相同格式的信息。ul是无序列表，ol是有序列表。

    dl, dt, dd表示一个自定义列表。

     <ul>
     <li>第一个无序列表</li>
     <li>第二个无序列表</li>
 </ul>
 <ol>
     <li>第一个有序列表</li>
     <li>第二个有序列表</li>
 </ol>
 <dl>
     <dt>ul</dt>
     <dd>无序</dd>
     <dt>ol</dt>
     <dd>有序</dd>
 </dl>

行内元素（inline element）

  • a

    一个超链接或者同一页面的锚点。href属性是要跳转的地址或锚点地址。

     <a href="http://www.google.com">google</a>
  • em, strong, del

    em是内容强调，strong表示比em更重要，del表示信息被删除。

     <p>
     <em>重要</em>
     <strong>更重要</strong>
     <del>删除的信息</del>
 </p>
  • span

    用来标识其他标签不适合表示的内容，是一个通用型的行内标签。可以使用它来编组元素。

     <p>
     <span>text</span>
 </p>
  • img

    代表一张图片，src属性是图片的地址。

     <img src="img/img.png" />
  • button

    表示一个可以点击的按钮。

     <button>submit</button>
